Dualverseは、画面分割で2つのSNSを同時に閲覧できる「究極のSNS廃人向けウェブブラウザ」です。
WebView2コンポーネントにより、ChromiumベースMicrosoft Edgeと同等のブラウジング性能を実現しました。
「リンクを新しいウィンドウで開く」が押された時は既定のウェブブラウザ (Firefox、Chrome、Edgeなど) を起動します。 

Dualverse ウェブサイト: https://arkw.net/products/win/Dualverse/
本ソフトウェアのソースコード: https://github.com/arkwnet/Dualverse/
Arakawa Laboratory ウェブサイト: https://arkw.net/

# 動作環境
OS: Windows 10 / 11
CPU: Intelまたは同等の互換プロセッサ 2GHz以上 (マルチコアプロセッサを推奨)
メモリ: 8GB以上
ストレージ: 1GB以上の空き容量
ディスプレイ: 1366×768ピクセル以上の解像度で色深度 32bit True color以上表示可能なもの
対応サービス: X (旧Twitter)、X Pro (旧TweetDeck)、Bluesky、Threads、Misskey.io、mstdn.jp、Pawoo、fedibird 他
